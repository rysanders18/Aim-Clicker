# Aim Clicker

## Overview
Aim Clicker is a hybrid game that combines the precision-based shooting mechanics of aim training games (such as Aim Lab) with the progression system of idle/clicker games (such as Cookie Clicker).

The player takes the role of a bounty hunter who earns money by shooting targets. The earned money can be used to purchase upgrades that generate passive income over time.

---

## Core Gameplay

### Active Gameplay (Aim System)
- Targets spawn in front of the player
- Player clicks on targets to destroy them
- Each hit rewards money
- Targets disappear after a short lifetime if not hit

### Idle / Clicker System
- Players can purchase upgrades using money
- Example: Auto Collector
  - Costs money
  - Provides passive income
- Passive income increases every second

---

## Features Implemented

- 3D target spawning system
- Raycast-based shooting system
- Target lifetime system
- Money system with UI display
- Passive income system
- Upgrade system (Auto Collector)
- Shop UI toggle (Press **Tab**)

---

## Controls

- Left Mouse Click → Shoot targets  
- Tab → Open / Close shop
